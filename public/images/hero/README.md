# Hero 背景图片说明

## 当前图片

当前使用的图片来自 Pexels（免费商用图片库）。

## 替换为更合适的图片

如果您想使用有中国人面孔的讲师和学生配图，建议：

### 推荐图片源

1. **Pexels** (https://www.pexels.com/)
   - 搜索关键词：`chinese teacher student`, `asian classroom`, `language learning`
   - 免费商用，无需署名

2. **Unsplash** (https://unsplash.com/)
   - 搜索关键词：`chinese language class`, `asian teacher`, `mandarin learning`
   - 免费商用

3. **Pixabay** (https://pixabay.com/)
   - 搜索关键词：`chinese teacher`, `language class`, `asian education`
   - 免费商用

### 图片要求

- **尺寸**：建议 1200x800px 或更大（横向）
- **格式**：JPG 或 PNG
- **内容**：有中国人面孔的讲师和学生一起学习的温馨场景
- **风格**：明亮、温馨、专业

### 替换步骤

1. 下载合适的图片
2. 将图片重命名为 `hero-main.jpg`
3. 替换 `public/images/hero/hero-main.jpg` 文件
4. 刷新浏览器查看效果

### 图片优化建议

- 使用图片压缩工具减小文件大小（建议 < 500KB）
- 确保图片清晰度足够
- 注意图片的构图，确保文字区域不会被重要内容遮挡
